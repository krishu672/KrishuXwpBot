## 🚀 Quick Deploy on Render

### Method 1: One-Click Deploy (Recommended)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/krishu672/KrishuXwpBot)

### Method 2: Manual Deploy

1. **Fork this repository** on GitHub
2. Go to [Render Dashboard](https://dashboard.render.com/)
3. Click **"New +"** → **"Web Service"**
4. Connect your GitHub repository
5. Configure:
   - **Name:** `krishuxwp-bot`
   - **Runtime:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `node index.js`
   - **Plan:** Free
6. Add Environment Variable:
   - `OWNER_NUMBER`: Your WhatsApp number (e.g., `919876543210`)
7. Click **"Deploy Web Service"**

### Pairing Code Method (No QR Code!)

1. After deploy, check the **Render Logs**
2. You'll see a **Pairing Code** displayed
3. Open WhatsApp on your phone → **Linked Devices** → **Link a Device**
4. Enter the pairing code shown in logs
5. Bot is connected! ✅

### Keep Bot Online 24/7

Use **cron-job.org** (free):
1. Go to [cron-job.org](https://cron-job.org)
2. Create account
3. Add new cron job
4. Set URL: `https://krishuxwp-bot.onrender.com`
5. Set interval: Every **14 minutes**
6. Save — bot stays online 24/7!

## 📱 Mobile Setup Guide

1. Create GitHub account (mobile browser)
2. Fork this repository
3. Go to render.com → Sign up with Google/GitHub
4. Click "New Web Service" → Connect GitHub
5. Select repository → Deploy (Free)
6. Check logs for pairing code
7. Link WhatsApp device
8. **Done!** Bot is running 24/7
